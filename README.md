# mp-tools

#### 介绍
一个用来在工作中提高生产力的插件


#### 安装教程

暂没有上传，可以自己打包本地安装

1. 下载项目，本地build（可能会很久），因为要下idea插件相关的sdk和gradle

2. idea-preference-plugins

#### 功能列表
1. 生成项目脚手架
2. 命名查询
3. mybatis-plus注解一键生成（开发中）
4. 逆向工程（规划中）
5. mapper service的mybatis-plus继承关系生成（规划中）
6. mapper service文件生成（规划中）
